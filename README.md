# Molehunt-Universal-Datapack
A minecraft datpack for the minecraft minigame Molehunt.

This minigame was created by [Piffin380](https://www.youtube.com/@Piffin380). I am simply a fan recreating the game in datapack form for others to enjoy.



## Thank you to these people for contributing to the pack

Bennerr and [Un1q32](https://github.com/Un1q32) for playtesting, fixes and a few other implementations.

[GalSergey](https://modrinth.com/user/GalSergey) for an implementation in Infection.

[Trevoman](https://modrinth.com/user/trevoman) for the implementation of the timer.
